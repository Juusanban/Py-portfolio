# Japan Area Code Finder

日本の都市名と市外局番を検索・管理する、Pythonのコンソールアプリケーションです。

Pythonの基本文法、CSVファイルの読み書き、入力チェックなどを学ぶために作成しました。

## 主な機能

* 都市名から市外局番を検索
* 市外局番から都市名を検索
* 登録データの一覧表示
* 新しい都市と市外局番の追加
* 登録済みデータの変更
* 登録済みデータの削除
* CSVファイルへの保存
* 市外局番の入力チェック

## 使用技術

* Python
* CSV
* Regular Expression
* pathlib

## ファイル構成

japan-area-code-finder/
├── main.py
├── city_codes.csv
└── README.md

## データ形式

`city_codes.csv`には、都市名と市外局番を保存しています。

例：

city,area_code
東京,03
名古屋,052
京都,075

## 実行方法

### 1. Pythonの確認

Pythonがインストールされていることを確認してください。

python --version

### 2. プロジェクトの実行

プロジェクトフォルダで、次のコマンドを実行します。

python main.py

## 操作メニュー

プログラムを起動すると、次のメニューが表示されます。

1. 都市名から検索
2. 市外局番から検索
3. 一覧を表示
4. データを追加
5. データを変更
6. データを削除
7. 終了

使用したい機能の番号を入力してください。

## 使用例

都市名から市外局番を検索する場合：

都市名を入力してください: 京都

実行結果：

京都の市外局番は075です。

市外局番から都市名を検索する場合：

市外局番を入力してください: 03

実行結果：

03の都市: 東京

## 入力チェック

市外局番は、次の条件で確認しています。

* 0から始まる
* 2桁から5桁の数字
* 正規表現を使用して形式を確認

同じ都市名がすでに登録されている場合は、新しく追加できません。
