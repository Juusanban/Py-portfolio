# Japan Area Code Finder

日本の都市名と市外局番を検索・管理するPythonのコンソールアプリです。

都市名から市外局番を検索したり、市外局番から都市名を検索したりできます。

## 主な機能

* 都市名から市外局番を検索
* 市外局番から都市名を検索
* 登録データの一覧表示
* 新しい都市と市外局番の追加
* 登録済みデータの変更
* 登録済みデータの削除
* CSVファイルへの保存
* 市外局番の入力チェック

## 使用技術

* Python
* CSV
* Regular Expression
* pathlib
* List
* Dictionary

## 使い方

1. プログラムを起動する
2. メニューから操作を選択する
3. 都市名または市外局番を入力する
4. 検索結果を確認する
5. 必要に応じてデータを追加、変更、削除する

## 実行方法

`python main.py`

## 実装内容

このアプリでは、都市名と市外局番のデータをCSVファイルで管理しています。

CSVファイルからデータを読み込み、都市名検索、市外局番検索、一覧表示、追加、変更、削除を行います。

市外局番は正規表現を使って形式をチェックしています。
