# Turtle Fractal Generator

Turtleを使用して、幾何学模様やフラクタル図形を描画するPythonアプリです。

回転する正方形、幾何学模様、コッホ曲線、コッホ雪片を描画できます。

## 主な機能

* 回転する正方形の描画
* 幾何学模様の描画
* コッホ曲線の描画
* コッホ雪片の描画
* 再帰レベルの選択
* 入力値チェック

## 使用技術

* Python
* Turtle
* Function
* for Loop
* Recursion
* Input Validation

## 使い方

1. プログラムを起動する
2. メニューから描画したい図形を選択する
3. コッホ曲線またはコッホ雪片では再帰レベルを入力する
4. Turtleウィンドウで描画結果を確認する
5. 描画後、ウィンドウをクリックして終了する

## 実行方法

`python main.py`

## 実装内容

このアプリでは、Turtleを使って図形を描画しています。

通常の図形はforループで描画し、コッホ曲線とコッホ雪片は再帰処理で描画しています。

再帰レベルを変更することで、図形の細かさを変えることができます。
