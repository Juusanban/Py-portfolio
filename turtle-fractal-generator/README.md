# Turtle Fractal Generator

Pythonの`Turtle`モジュールを使用して、幾何学模様やフラクタル図形を描画するコンソールアプリケーションです。

Pythonの関数、繰り返し処理、座標、角度、再帰処理を学ぶために作成しました。

## 主な機能

以下の4種類の図形を描画できます。

1. 回転する正方形
2. 幾何学模様
3. コッホ曲線
4. コッホ雪片

コッホ曲線とコッホ雪片では、再帰レベルを0から5まで選択できます。

## 使用技術

* Python
* Turtle
* 再帰処理
* 関数
* `for`ループ
* 入力チェック

## 実行方法

### 1. Pythonの確認

Pythonがインストールされていることを確認してください。

python --version

### 2. プログラムの実行

プロジェクトフォルダで、次のコマンドを実行します。

python main.py

## 操作方法

プログラムを起動すると、次のメニューが表示されます。

1. 回転する正方形
2. 幾何学模様
3. コッホ曲線
4. コッホ雪片

描画したい図形の番号を入力してください。

## 図形の説明

### 1. 回転する正方形

正方形を少しずつ回転させながら、36回描画します。

`for`ループと角度の変更を使用しています。

### 2. 幾何学模様

線を描画しながら一定の角度で回転し、幾何学的な模様を作成します。

### 3. コッホ曲線

1本の直線を細かく分割し、再帰処理を使用してフラクタル図形を描画します。

### 4. コッホ雪片

3本のコッホ曲線を組み合わせて、雪の結晶のような閉じた図形を描画します。

## 再帰レベル

コッホ曲線とコッホ雪片を選択した場合、再帰レベルを0から5まで入力します。

* レベル0：通常の直線
* レベル1：大きな凹凸を追加
* レベル2：より細かい凹凸を追加
* レベル3：標準的なフラクタル図形
* レベル4：細かい図形
* レベル5：非常に細かい図形

再帰レベルが高くなるほど、描画される線の数が増えます。

## 使用例

コッホ雪片を描画する場合：

描画する図形を選択してください: 4

再帰レベルを入力してください（0〜5）: 3

入力後、Turtleのウィンドウが開き、図形が描画されます。

描画が終了した後、ウィンドウ内をクリックすると終了します。

## 入力チェック

以下の入力チェックを行っています。

* 図形の選択は1から4まで
* 再帰レベルは0から5まで
* 再帰レベルに数字以外が入力された場合は再入力
