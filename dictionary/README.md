# Dictionary Builder

## 開発状況

## 概要
PythonとTkinterを使って作成した単語帳アプリです。  
単語の追加・更新・削除、検索、JSON保存、Excel出力、Anki出力に対応しています。

## 実装済み機能
- 単語カードの追加
- 単語カードの更新
- 単語カードの削除
- 単語検索
- JSON形式での保存
- Excelファイルへの出力
- Anki用ファイルの出力
- 英語・日本語タブの切り替え
